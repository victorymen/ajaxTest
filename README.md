# ethereum-slot-machine
基于以太坊的水果机游戏V2.0

# 效果
![](./demo.gif)

# 机器环境
- win10
- nodev8.9.4

# 安装依赖
- npm install -g truffle
- npm install -g ganache-cli

# 项目的运行
- （1）启动ganache-cli：ganache-cli
- （2）迁移合约：truffle migrate
- （3）npm install && npm run dev
